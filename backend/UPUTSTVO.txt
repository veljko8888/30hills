Postovani,
API je realizovan u sledecim tehnologijama: PHP, HTML, CSS, MySQL(baza).
Odradjen je backend test sa sve tri funkcionalnosti.

1) 30hills.sql file importovati kao novu bazu pod nazivom 30hills(username: root, pw: )
2) na serveru pokrenuti file home.php
3) popuniti bazu klikom na dugme Put data into database
4) testirati aplikaciju/api

PS: u prilozenom data.json file-u koji ste poslali postoji greska:
user sa ID=16 ima za prijatelja 18 i 20, medjutim user sa ID=18
u svom nizu prijatelju nema usera sa ID=16 sto je kontradiktorno,
ako je 18 prijatelj useru 16, onda i user 16 mora biti prijatelj useru 18.
